## Authentication

All endpoints in the Chatbot Builder AI API are authenticated using an API key passed in the header of your requests.

To authenticate your requests, you need to include an `X-ACCESS-TOKEN` header with your unique API key.

```json
"security": [
  {
    "APIKeyHeader": []
  }
]
```
